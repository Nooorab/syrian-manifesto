ABOUT
-----
This is the official website for "بيان من أجل يقظة سوريا" (A Manifesto for 
Syria's Awakening). The website presents a comprehensive manifesto addressing 
the spiritual and psychological healing needed for Syrian society after 
decades of oppression.


FEATURES
--------
• Interactive manifesto statements with expandable details
• Commitment pledge counter
• Guestbook for visitor messages
• Responsive design for all devices
• Arabic RTL (right-to-left) layout
• Modern glassmorphism design aesthetic


TECHNICAL DETAILS
-----------------
• Built with: HTML5, CSS3, JavaScript
• Fonts: Amiri, Cairo (Google Fonts)
• No external frameworks required
• Fully responsive design


HOW TO RUN
----------
Option 1: Open index.html directly in a web browser

Option 2: Run a local server
    - Navigate to the project folder in terminal
    - Run: python3 -m http.server 8000
    - Open http://localhost:8000 in your browser
